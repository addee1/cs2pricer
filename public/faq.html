Why do I need to use my steam API:key ?
Your API key is used to securely fetch information such as your inventory details from the platform. It ensures authorized and personalized access. Importantly, the API key is not stored for added security. Never share your key to anyone!
Why do I need to use my SteamId ?
For developers and third-party services that need to gather information or interact with Steam's API, a Steam ID is essential to direct requests to specific users.
Why is my inventory not loading ?
Ensure that your Steam profile is public and that you have entered the correct SteamID64 and API key. If the problem persists, please try again later.

You need your SteamID64 & API-key. You can find each here: API-Key SteamID64. If you don't already have an API-key you need to create one. Enter your SteamID64 and API key.
